## Descrição do Projeto
Este projeto consiste no desenvolvimento de um jogo Batalha Naval utilizando o framework Flutter e a linguagem Dart. O jogo é uma versão digital do clássico Batalha Naval, o jogador deve localizar e destruir todos os navios no lado do tabuleiro inimigo, tentando acertar a coordenada do navio inimigo


## Funcionamento do Jogo
O jogo utiliza um tabuleiro em formato de grade 16X16, os navios sao posicionados pelo jogador na sua parte do tabuleiro. Depois das peças posicionadas o jogador inimigo escolhe onde quer atacar. Apos a tentativa o sistema informa se você acertou ou errou um navio. E como no jogo original sua missão é destruir o navio inimigo


## Tecnologias Utilizadas
O projeto foi feito em Flutter com Flame , e tambem foi usado Dart

## Como Executar o Projeto
Digite flutter run no terminal dentro da pasta do projeto para rodalo
